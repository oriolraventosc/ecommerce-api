# E-commerce API

## Endpoints

### **GET**

#### /products/loadAll

#### Load all the products of the database

### **PATCH**

#### /products/checkout

#### Checkout all the products of the cart

### **POST**

#### /admin/login

#### Login as a administrator

### **GET**

#### /admin/pending-orders

#### Load all the pending orders

### **PATCH**

#### /admin/accept-order/:id

#### Accept a order

### **PATCH**

#### /admin/cancel-order/:id

#### Cancel a order
